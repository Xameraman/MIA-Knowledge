WHAT IS NOVA

NOVA stands for Nexus Of Verified Assets. NOVA is a community‑driven Discord server and network of guide websites built around fast, verified, direct downloads for Minecraft Bedrock Edition content. NOVA was founded and is owned by Xameraman5022. Every file shared in NOVA goes through a structured system — categories are organized, credits are required, links are verified direct downloads, and no ad links or shorteners are ever allowed. NOVA always credits the original creator of every pack and respects creator requests immediately. NOVA has a Discord server, 12 official guide websites, an online library at nova‑library.pages.dev, the ARIA AI helper bot, a YouTube channel, and a GitHub‑powered database. The NOVA Discord server has approximately 179 members and new content is shared daily. NOVA is also listed on Disboard for public discovery.

HOW TO JOIN AND USE NOVA

1. Join the NOVA Discord server using the permanent invite link: https://discord.gg/ThgxvPtUjZ
2. Once in the server, read the rules channel first. NOVA uses Wick bot for moderation. English is the server language.
3. Get the Key Access role to unlock most channels. This is the verification role.
4. Browse the 22 organized content channels: PvP, Utility, Furniture, Horror, Anime, Shader, Structure, Weapon, Bosses, Vehicle, Visual, Survival, Client, GUI & UI, Adventure, Minigame, Food, Environment, Addon, World, Animation, Skins.
5. To download content, click the direct links shared in each channel. Links go to trusted file hosts like MediaFire, MEGA, Google Drive or Proton Drive. There are never any ad links or Linkvertise wrappers in NOVA.
6. If you need help with a download, visit the NOVA guides hub at https://xameraman.github.io/all-nova-guides/ and pick the guide for the file host you are using. NOVA has 12 guides covering: Java & Bedrock Import, MCPEDL, MediaFire, Mirror, CurseForge, Modrinth, MEGA, Linkvertise, Proton Drive, Sub2Unlock, Urlspay and the All Guides hub.
7. If you are a creator uploading to NOVA, use direct links only. For Bedrock content use .mcpack or .mcaddon files. For Java content use .jar or .zip files. Always credit the original creator. Images are mandatory for all submissions.

HOW NOVA UPLOADS WORK

Packmakers who create their own content use the **/share-with-nova** command. It requires name, a direct file link, category, credits, support link, an archive preference (YES, NO, IDK, or a personal permanent link — filego.io is **never** used for archives), and a preview photo. The submission is reviewed by staff, and if approved the packmaker gets the Verified Packmaker role and the pack is featured by an Experienced NOVA Sharer using **/nova-feature**.

NOVA Items Sharers share outside creator content using **/nova-share**. This command takes the pack name, a direct download link, original creator credit, a short description, a preview image link, the correct category, and an optional support link. The bot sends a formatted embed with a 🚀 Get It Now button and, if a valid support link is provided, a 💝 Support The Creator button.

Experienced NOVA Sharers feature verified packmaker submissions with **/nova-feature**. This command adds a permanent archive link (never filego.io) and can produce up to three buttons: 🚀 Get It Now, 🌌 Temp Mirror, and 💝 Support The Creator.

After any announce command, the announcer must publish both the bot text message and a separate preview photo in the channel, in that order, to the connected server.

All uploads must use direct download links. No Linkvertise, no Urlspay, no Sub2Unlock wrappers. Every upload must credit the original creator. Images are mandatory.

COMMON PROBLEMS

PROBLEM: I cannot access channels in NOVA Discord.
SOLUTION: Get the Key Access verification role. Check the rules channel for instructions.

PROBLEM: A download link is broken or expired.
SOLUTION: Report it using the **/nova-report** command in the server, or ask in the channel where it was posted.

PROBLEM: I do not know how to install a downloaded file.
SOLUTION: Visit the NOVA Java & Bedrock Import Guide at https://xameraman.github.io/guide-for-java-and-bedrock/

PROBLEM: I see a Linkvertise or ad link in NOVA.
SOLUTION: NOVA never allows ad links. Report the message immediately with /nova-report or notify a staff member.

PROBLEM: I want to upload but do not know the commands.
SOLUTION: Packmakers use **/share-with-nova**. Sharers use **/nova-share**. Experienced sharers use **/nova-feature**. Check the server instructions channel for detailed help, or ask ARIA.

PROBLEM: The ARIA bot is not responding.
SOLUTION: Wait a few seconds and try again. If it still fails, ask in a support channel or contact a staff member.
